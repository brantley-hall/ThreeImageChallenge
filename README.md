**July2026Version1** - NYU from Washington Square Park Forensic Image Analysis

## Overview

This is a forensic image analysis tool for examining historical photographs of NYU from Washington Square Park. The application allows users to overlay historical maps and photographs with modern perspectives to analyze architectural changes and historical accuracy.

## Features

- **Interactive Map Selection**: Choose from multiple historical maps (1852-2026)
- **Overlay System**: Compare different historical photographs and maps
- **Camera Controls**: Adjust camera position, height, rotation, and field of view
- **Building Information System**: View detailed information about NYU buildings and surrounding structures
- **Real-time Rendering**: Canvas-based visualization with adjustable opacity and positioning

## Historical Maps Included

- 1852 Harrison Map

- 1870 NYU Collection

- 1890 NYU Archive

- Austen 1894

## Technical Details

- **Version**: July2026Version1
- **Framework**: Pure HTML5, CSS3, and JavaScript
- **Rendering**: HTML5 Canvas
- **Data**: LocalStorage for settings persistence
- **Compatibility**: Modern browsers with Canvas support

## Usage

1. Select a historical map from the sidebar
2. Adjust camera settings to match the photograph perspective
3. Overlay additional historical images for comparison
4. Use building controls to explore architectural details
5. Save and load different perspective configurations
