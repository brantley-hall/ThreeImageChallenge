# ThreeImagesChallenge

A forensic image analysis tool for examining historical photographs of NYU from Washington Square Park, c. 1839-1895.

## Overview

This interactive web application allows users to analyze historical daguerreotypes and photographs of the original NYU Gothic Revival building using geometric reconstruction and camera positioning tools.

## Features

- **Interactive Map System**: Switch between 6 historical maps (1852-1893)
- **Camera Positioning**: Adjust camera location, bearing, and field of view
- **Building Analysis**: Position NYU building, row houses, and Church of Messiah
- **Overlay Comparison**: Compare historical photographs with 3D reconstructions
- **Plate Size Options**: Support for various daguerreotype plate formats

## Historical Maps

1. **Site Plan 1852 Harrison** - Original survey map
2. **1854 Perris Map Extract** - Detailed neighborhood survey
3. **2026 NYC Zoning** - Modern zoning map for reference
4. **1879 Brumley Map** - Bromley atlas map
5. **1891 NYU Bromley** - NYU-specific Bromley map
6. **1893 Pincus Map** - Lionel Pincus and Princess Firyal Map Division

## Overlay Photographs

- **Plate H** - Original daguerreotype (c. 1839)
- **King's Handbook 1893** - Published photograph
- **NYU Collection 1893** - Archive photograph
- **NYU Collection 1870** - Early archive photograph
- **NYU from Middle of Park** - Alternative 1891 view
- **Austen 1894** - Late 19th century photograph
- **1839 Smithsonian** - Church of Messiah photograph

## Usage

1. **Select a Map**: Choose from the dropdown menu or thumbnail grid
2. **Position Camera**: Use the controls to set camera position and field of view
3. **Adjust Buildings**: Move NYU building, row houses, and church to match historical photographs
4. **Compare Overlays**: Select historical photographs to overlay on the 3D view
5. **Analyze Sight Lines**: Use the viewfinder to analyze what was visible from different positions

## Technical Details

- **Plate Sizes Supported**: Quarter plate (4.25×3.25), 2×2.5, 5×7 portrait/landscape
- **Field of View**: Adjustable from 2° to 60° for realistic telephoto simulation
- **Coordinate System**: Standard X,Y positioning with rotation controls
- **Image Loading**: Automatic fallback system for reliable map display

## Deployment

This application is designed for GitHub Pages deployment. All assets are self-contained and the application runs entirely in the browser using HTML5 Canvas and JavaScript.

## Research Purpose

This tool supports historical research by enabling:
- Geometric verification of photograph authenticity
- Reconstruction of camera positions from historical descriptions
- Analysis of building sight lines and visibility
- Comparison of different photographic techniques and eras
